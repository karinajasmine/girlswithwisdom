    Java

    Java is a programming language that creates softwares for a multiplatude of platforms. This language is an Object Oriented programming language. The goal of this language that code would be able to run inside of web browsers. This language is known to be by far the most popular and used language. Once learning this language you are able to apply your knowledge onto other Object Oriented languages, such as JavaScript, C#, and C++. By choosing this language girls would be paired with experianced coders who are fluent in Java and compile an abundance of codes to create their very games.

    Python

    Python is a programming language that is targeted for web applications, software applications, GUI applications, pages within webs, and creating games. Python is known to typically be a high level programming language, however it is fairly simple after learning the syntactical specifics. Python is a scripting language. Python can also be utilized for data analysis and visualizations. By choosing to study this language one would be paired with other girls who are passionate about python and be able to program their own codes and even create their own games.

    HTML CSS & JavaScript

    HTML (HyperText Markup Language), CSS (Cascading Style Sheets, and JavaScript are the main programming languages when it comes to programming your very own website and designing it. All of these languages are fairly simple and are different in comparison to python and Java. HTML is the foundation of the website and creating raw content. CSS is meant for formatting and designing the website for a presentable outcome. Finally, JavaScript is for content and formatting. These languages go hand in hand and are very useful for displaying peoples passions or even creating your very own startup! By picking to learn about HTML, CSS, and JavaScript one would learn how to create their very own website and if they would like publish it for the web!
